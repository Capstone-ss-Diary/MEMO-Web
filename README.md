## 개요
'MEMO'는 온·오프라인 일기의 장점을 결합시킨 형태의 개인 기록용 다이어리 웹 서비스입니다. 일기의 작성과 편집은 물론, 자동 해시태그 생성, 사용자 손글씨 적용 등의 기능을 제공하며 일기에 관련한 다양한 스티커들을 추천해주며 오프라인 못지 않은 다이어리 꾸미기 기능을 제공합니다.
